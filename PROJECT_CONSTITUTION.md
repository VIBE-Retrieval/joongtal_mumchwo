# Project Constitution

## Project Overview

This document defines the foundation and governance structure of the project.

## Layers

### Layer 1: Constitution

- **PROJECT_CONSTITUTION.md** - Project foundation and governance

### Layer 2: Agents

- **AGENTS/** - Agent specifications and orchestration
  - orchestrator.md - Main orchestrator agent
  - frontend-agent.md - Frontend agent specifications
  - backend-agent.md - Backend agent specifications
  - ai-agent.md - AI agent specifications
  - db-agent.md - Database agent specifications
  - docs-agent.md - Documentation agent specifications

### Layer 3: Implementation & Documentation

- **frontend/** - Frontend implementation
- **backend/** - Backend implementation
- **AI/** - AI/ML implementation
- **DB/** - Database implementation
- **Docs/** - Project documentation
  - architecture/ - Architecture documentation
  - api/ - API documentation
  - db/ - Database documentation
  - ml/ - Machine Learning documentation
  - llm/ - LLM documentation
  - agent/ - Agent documentation
  - ux/ - UX/Design documentation

## Project Structure
